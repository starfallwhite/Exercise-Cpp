# Exercise-Cpp
C++练习案例
1.演讲比赛
    * 学校举行一场演讲比赛，共有**12个人**参加。**比赛共两轮**，第一轮为淘汰赛，第二轮为决赛。
    * 比赛方式：**分组比赛，每组6个人**；选手每次要随机分组，进行比赛
    * 每名选手都有对应的**编号**，如 10001 ~ 10012 
    * 第一轮分为两个小组，每组6个人。 整体按照选手编号进行**抽签**后顺序演讲。
    * 当小组演讲完后，淘汰组内排名最后的三个选手，**前三名晋级**，进入下一轮的比赛。
    * 第二轮为决赛，**前三名胜出**
    * 每轮比赛过后需要**显示晋级选手的信息**
